# TechStore E-commerce

Dự án Website bán hàng công nghệ Fullstack.
Demo: [Bấm vào đây để xem trang web](https://project-ecommerce-techstore.vercel.app)

## Công nghệ sử dụng
- **Frontend:** React (Vite), TypeScript.
- **Backend:** NestJS, TypeORM/Prisma.
- **Database:** PostgreSQL (Neon Tech).
- **Deploy:** Vercel (FE) & Render (BE).

## Tính năng chính
- Xem danh sách sản phẩm.
- Đăng ký / Đăng nhập (JWT).
- Admin Dashboard (Thêm sửa xóa sản phẩm).
- Upload ảnh (Cloudinary).
